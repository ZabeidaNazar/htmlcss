Html

Теги записуються у <>

Html теги бувають:
одинарні: <br>, <img>
парні: <html></html>, <body></body>, <h1></h1>, <p></p>, <a></a>

Парні теги складаються з відкриваючого <тег> (<title>) і закриваючого </тег> (</title>) тегів.

Одинарні теги складаються тільки з відкриваючого тегу.

Атрибути записуються до відкриваючого тегу через пробіл, значення атрибуту записується після знаку = в лапках "":
<тег атрибут=""></тег>
Атрибутів може бути декілька.

Всі теги мають атрибут style, за допомогою якого можна задавати Css стилі, як значення атрибуту.

Всі теги мають атрибут class, за допомогою якого можна задавати Css стилі за допомогою css файлу.

Також всі теги мають атрибут id, за допомогою якого можна створити посилання на частину сайту.
id тегів повинні бути унікальними, вони не мають повторюватися!

Наприклад, в нас є абзац зі статтею, проте він знаходиться в самому кінці.
<p>Стаття</p>
На початку сайту ми хочемо зробити зміст, який матиме посилання на цю статтю.
<h3>Зміст</h3>
Додамо до тегу <p> атрибут id зі значенням, наприклад, last-article.
<p id="last-article">Стаття</p>
Після цього скористаємося тегом <a>, вказавши в посиланні символ #, а тоді id елемента, на який хочемо переходити.
<h3>Зміст</h3>
<a href="#last-article">Остання стаття</a>
Тепер при натисканні на текст "Остання статя" ми будемо переходити до абзацу, id якого last-article.

p Абзац тексту —

b Жирний текст —

table Таблиця —

Одинарні теги

Тег Призначення Атрибути
img Вставлення зображення src — посилання на зображення
width — ширина зображення
height — висота зображення
opacity — прозорість (від 0 до 1)
br Переносить на наступний рядок —
link Підключення css, шрифтів і тд. rel="stylesheet" — для підключення css
href — посилання на елемент, який підключаємо

*Теги мають дуже багато атрибутів, позначені тільки ті, які вивчалися!

**Атрибути id і style не зазначені, оскільки їх мають усі теги.
