Css

Css стилі можна задавати кількома способоми:

  • Записувати в атрибуті style: стиль: значення; стиль: значення
  • Записувати в парному тегу style
  • Записувати в файлі з розширенням .css

Останній спосіб найкращий, але потрібно не забути підключити css файл до файлу html.
Для цього необхідно скористатися командою <link rel="stylesheet" href="<шлях до css файлу>">
